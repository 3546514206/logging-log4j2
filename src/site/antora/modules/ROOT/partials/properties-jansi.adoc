[[log4j2.skipJansi]]
== `log4j2.skipJansi`

[cols="1h,5"]
|===
| Env. variable | LOG4J_SKIP_JANSI
| Default value | ``true``
|===

If the following conditions are satisfied:

* Log4j runs on Windows,
* this property is set to `false`,

Log4j will use the JANSI library to color the output of the console appender.
